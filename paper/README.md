# paper/

The write-up of the lambda-lattice study lands here (LaTeX source + built PDF).

Until then, the complete scientific record lives in:

- `../README.md` — the results narrative, tables, and figures.
- `../preregistrations/` — the three pre-registration ledgers (each committed
  before its run), including the amendment histories and outcome verdicts.
- `../data/` — every raw per-probe result JSON reported anywhere.
- `../figures/` — F1–F10 (regenerable with `lambda-lattice figures`).

When the article is added, this directory will also hold the exact
`data/`→figure→table build so every number in the paper is reproducible from the
shipped JSONs.
